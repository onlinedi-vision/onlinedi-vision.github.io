POST /api/accept_dm_invite

Accept a DM invite.

Parameters

Parameter Type Required Description
token string Yes Recipient's authentication token.
recipient string Yes Recipient's username.
sender string Yes Sender's username. 
                      
curl -X POST 'https://onlinedi.vision/api/accept_dm_invite' \
-H 'Content-Type: application/json' \
-d '{ 
    "token": "valid_token", 
    "recipient": "username2"
    "sender": "username1"
}' 



{
    "status": "dm_created", 
    "sid": "!generated_sid", 
    "invite_id": "uuid_invite", 
    "u1": "username1", 
    "u2": "username2", 
    "sender": "username1"
}
POST /api/add_server_role

Add a new role to a server.

Parameters

Parameter Type Required Description
token string Yes User authentication token.
username string Yes User's unique username.
server_id string Yes Id of the server that will receive this new role.
role_name string Yes Name of the new role.
color string Yes Color code for the new role.
permissions collection Yes Collection of permissions that will be asigned to the role.

-H 'Content-Type: application/json' \
-d '{ 
    "token": "valid_token", 
    "username": "username123", 
    "server_id": "valid_id_server", 
    "role_name": "Moderator", 
    "color": "#FF0000", 
    "permissions": "['kick_users', 'ban_users']"
}' 



{
   "Role added successfully."
}

POST /servers/{sid}/api/{channel_name}/delete_message

Deletes a specific message from a given channel, based on its UTC timestamp.

Parameters

Parameter Type Required Description
datetime string Yes UTC timestamp of the message to be deleted (format: YYYY-MM-DD HH:MM:SS.sss).
token string Yes User's authentication token.
username string Yes Username of the user requesting message deletion.
sid string Yes Server ID where the channel is located.
channel_name string Yes Name of the channel containing the message. 
                      
curl -X POST 'https://onlinedi.vision/servers/{sid}/api/{channel_name}/delete_message' \
-H 'Content-Type: application/json' \
-d '{
    "datetime": "2025-10-18 11:55:10.009",
    "token": "valid_token",
    "username": "user123"
}'



"Message deleted successfully"
