[od-official-server] API Documentation

Comprehensive guide to using our RESTful API endpoints.

Endpoints

  • /servers/{sid}/api/get_server_info
  • /api/try_login
  • /servers/{sid}/api/get_server_users
  • /api/new_user
GET /servers/{sid}/api/get_server_info

Get all relevant information about a public server.

Parameters

Parameter Type Required Description
sid string Yes The ID of the server. 
                      
curl -X GET 'https://onlinedi.vision/servers/1313/api/get_server_info'  



{
   "name":"Awsome Server"
   "desc":"This is an AWESOME server :3"
   "img_url":"https://link.to.image"
}
POST /api/try_login

Log user in. If username-password combination is corect the return payload should contain a token.

Parameters

Parameter Type Required Description
username string Yes Username of the user.
password string Yes Password of the user. 
                      
curl -X POST 'https://onlinedi.vision/api/try_login' \
-H 'Content-Type: application/json' \
-d '{ 
    "username": "username123", 
    "password": "STRONGpassword" 
}' 



{
   "token": "hu345tdvnfjsui..."
}
POST /servers/{sid}/api/get_server_users

Get the list of users from a given server based on a valid token-username combination.

Parameters

Parameter Type Required Description
username string Yes Username of the user (must be member of the server).
email string Yes Token of the user (must be valid).
sid (in URL path) integer Yes Server ID of the queried server. 
                      
curl -X POST 'https://onlinedi.vision/servers/1313/api/get_server_users' \
-H 'Content-Type: application/json' \
-d '{ 
    "username": "username123", 
    "token": "asdfq3reQW355f..." 
}' 



{
   "u_list":[
        "username1",
        "username2",
   ]
}
POST /api/new_user

Create new user account.

Parameters

Parameter Type Required Description
username string Yes Username of the new user.
email string Yes E-mail address of the new user.
Password string Yes Salted password of the new user. 
                      
curl -X POST 'https://onlinedi.vision/api/new_user' \
-H 'Content-Type: application/json' \
-d '{ 
    "username": "username123", 
    "email": "username@example.com", 
    "password": "SALTEDsecurepassword123" 
}' 



{
   "token":"asdsaerggreasfd..."
}
